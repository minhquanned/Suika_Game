# Suika Game
Link play: [Suika Game](https://minhquanned.github.io/Suika_Game/)
#
Guide:
- Press 'R' to restart
- Press 'M' to mute
- Press ⬅️ and ➡️ or use your mouse to move
- Press 'Space' / 'Enter' / 'Left click' to drop
