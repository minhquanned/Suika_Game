# Suika Game
Clone the git and run _**suika_clone.html**_ file with _**live server**_.
#
Guide:
- Press 'R' to restart
- Press 'M' to mute
- Press ⬅️ and ➡️ or use your mouse to move
- Press 'Space' / 'Enter' / 'Left click' to drop
